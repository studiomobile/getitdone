Client start with specifying service request. For example: "I need some plumbing work done at my house" or "loundry services required" or "buy bread and almond milk"

If service require some qualifications like fixing electricity require electrical skills then client specify these requiremens.

Each service request require at least one location. Some services might require more, for example shopping services might require list of shops to make purchases.

Once service request published system select a helper according to location and available and send ETA back to client.

Services are billed on a hourly basis. Once service performed helper bill client according to number of hours spent on service.
